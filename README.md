# PFF 2.0 - Web Application Project

A web-based application developed by FantasticFurries in partial fulfillment for the requirements for Web Development Tools (WEBTOOLS), Web Applications Development (WEBAPPS), and Information Management (INFOMAN) class at the University of Makati. The project aimed to provide a functional system for browsing products, appointment scheduling, managing data and improving user experience in a web environment.

---

## Key Features
* **User Authentication:** Login and registration system
* **Database Management:** Full integration with MySQL for storing and retrieving records.
* **Appointment Management System** Fully integrated schedule appointment and management system.
* **Responsive Design:** The UI is designed to be accessible on different screen sizes.
* **Admin Dashboard:** A central place to manage files and user reports.
